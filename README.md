# quickstyle.com.au meteor + react app

## Run on local machine

`npm run dev`

`npm install` for the first time (don't use yarn, Meteor doesn't work well with yarn)

## Deploy on Heroku

`npm run deploy`

make sure heroku cli is installed and signed in
